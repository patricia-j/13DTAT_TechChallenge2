# 13DTAT_TechChallenge2
Olá!
Este repositório é para a atividade de Tech Challenge - Fase 2, da turma DTAT13.

A estrutura do repositório é a seguinte:
- data: contém o dataset utilizado
- notebooks: contém o arquivo com a análise exploratória dos dados e classificação dos modelos de machine learning
- results: contém o relatório executivo em PDF e um arquivo .txt com o link do vídeo
- requirements.txt: arquivo com as bibliotecas utilizadas
